# MVP Screen Scope

Launch target: July 31, 2026.

The original brief recommends 10-12 weeks. This scope compresses the delivery into a launchable MVP by preserving the full editorial loop and simplifying optional engagement features.

## Public Site

Home:

- Lead story, latest articles, category access, Featured/Media preview, contributor highlight.

About:

- Mission, vision, organizational overview, links to people pages.

Creative Team:

- Team profile cards with name, role, image placeholder, bio, social links.

Contributors:

- Contributor profile cards, published work links, social links.

Visceral Mag/articles:

- Article list, category chips, search entry, latest and featured editorial items.

Article detail:

- Title, dek, author, category, date, featured image, body, related articles, comments/reviews.

Categories/search:

- Category browsing, keyword search, filters, no-results state.

Featured/Media:

- Photography, artwork, media stories, captions, credits, alt text.

Contact:

- Contact form, validation, success/error states, spam protection note.

## Admin

Admin dashboard:

- Publishing statistics, pending moderation, recent activity, quick actions.

Article management:

- Article table, draft/publish workflow, SEO fields, rich body field, delete confirmation.

Profile/media management:

- Contributor and Creative Team management, media library, image metadata.

Comments/reviews moderation:

- Pending, approved, rejected queues with approve, reject, delete, search, filter.

Contact submissions:

- Inbox, detail view, new/read/archived states, search and filter.

## Deferred

- Newsletter campaigns
- Public user accounts
- Subscriptions
- Likes/bookmarks
- Advanced analytics dashboard
- Complex media hub
- Mobile app
