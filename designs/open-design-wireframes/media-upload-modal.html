Media Upload Modal wireframe

Admin media upload and insert modal with required accessibility metadata.

①

Media upload is an admin modal/subflow launched from Profile / Media Management and article editing.

②

Modal anatomy: upload/drop zone, preview, title, alt text, caption, credit, and category.

③

Required metadata includes usage, publish readiness, article link, rights note, and accessibility fields.

④

States cover uploading, missing alt text, invalid file, upload failure, and successful insert.

⑤

Actions include save draft metadata, insert media, cancel, and focus return to launcher.

Pin selection is interactive for review.
ModalTrap focus, Escape closes, return focus to launcher.
