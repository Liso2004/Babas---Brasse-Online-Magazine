Profile/Media Management

Contributor/team management plus media library metadata and upload/select flow.

①

Admin nav route groups profiles and media because both feed public profile and article views.

②

Contributor and Creative Team management show active/draft status, role, profile completeness, and edit action.

③

Media library card/table shows image, alt text, caption, credit, category, usage, and publish readiness.

④

Upload/select placeholder must collect alt text, caption, credit, and category before media can be public.

⑤

Uploading, missing metadata, file error, empty library, and permission states are required.

Pin selection is interactive for review.
ResponsiveProfiles become stacked management rows; media grid moves 3-up to 1-up.
A11yUpload dropzone must have keyboard path and labelled file input fallback.
StateBlock publish if media lacks alt text or credit where required.
